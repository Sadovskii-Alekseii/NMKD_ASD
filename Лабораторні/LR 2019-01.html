Лабораторна робота № 1.
Знайомство з Visual Studio. Пошук мінімального і максимального значення в масиві

Мета роботи

Ознайомитись з IDE Visual Studio. Навчитися створювати проект. Здобути навички у знаходженнi мінімального і максимального значення в масиві

Обладнання

персональний комп’ютер, Visual Studio 2008 або новіше

Хід роботи

  1. Завантажити програму Visual Studio 2008

    Знайдіть на робочому столі ярлик з Visual Studio 2008 або Пуск → Всі програми→ Microsoft → Microsoft Visual Studio 2008. Ознайомтеся з інтерфейсом

  2. Створити новий проект «Visual C++ (консольное приложение Win32)».

    Для того щоб створити новий проект в головному меню виберемо Файл → Cтворити → Проект.

    Далі виберемо мову програмування «Visual C++» і тип проекту «Консольное приложение Win32». Введемо ім'я проекту і вкажемо розташування файлів проекту. Натиснемо «ОК», в наступному вікні необхідно натиснути «Готово».

  3. Зберегти файл з іменем index.html

  4. У файл index.html помістити наступні рядки:
  5. Перевірити даний файл у валідаторі розмітки W3C за посиланням https://validator.w3.org/

  6. Виявити наявність помилок та попереджень. Скорегувати документ відповідно до рекомендацій у посиланнях 4 та 5 (тег <!DOCTYPE html>, вказати мову документу, кодування документу)

  7. Зберегти файл та перевірити його знову
  8. Profit! Ви створили першу валідну HTML-сторінку
  9. Створити короткий опис улюбленого фільму, книги, або комп'ютерної гри
  10. Використати теги заголовків, жирного тексту, абзацу, переліків та гіперпосилань, спираючись на довідники 2, 3, 6 та 7 - 14
  11. Внести зміни у файл index.html, зберегти його та перевірити через валідатор
  12. HTML-код файлу та скріншот перевірки валідатором зберегти у звіт
  13. Роздрукувати звіт та письмово відповісти на контрольні запитання
